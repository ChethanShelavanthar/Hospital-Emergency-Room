This project contains a dashboard analyzing hospital emergency room data.

## Metrics Included
- Number of Patients
- Average Wait Time
- Patient Satisfaction Score
- Patient Attendance Status
- Gender Analysis
- Age Group Distribution
- Department Referral Analysis

    Excel
